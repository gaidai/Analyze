Analyze  Specification
App created with:
- NetBeans 8.2
- Windows 7
- Java JDK 8.131
Manual:
1. Put Analyze.jar at one directory with two HTML files
2. Start the program with two String arguments for each file (file name or full URL of file) from console.
3. Change expression at the method parseOriginal for searching another tag (Analyze.java).
4. comparing process use's tag attributes , innerText, tagName.
5. Get result path of the much-closest analog in console .
